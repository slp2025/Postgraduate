# 🎓 Postgraduate

> 用于系统性收藏与整理本人在 **研究生阶段** 的学习笔记、课程资料与科研产出。

---

## 📌 仓库简介

本仓库主要用于记录与归档本人在读研期间的各类产出，包括但不限于：

- 📄 课程学习笔记与总结  
- 📊 课程作业 / 实验报告  
- 📐 数学与信号处理相关推导整理  
- 📡 通信与 5G / NR 物理层相关资料  
- 🧪 仿真代码、验证脚本与说明文档  
- 📚 文献阅读笔记与阶段性总结  

内容以 **自用、复盘、可追溯** 为主要目的，力求结构清晰、逻辑完整，便于后续查阅与扩展。

---
## 🎯 使用目的

- 📚 **知识沉淀**：将零散学习内容系统化、结构化  
- 🔁 **快速复盘**：为考试、开题、中期与答辩提供参考  
- 🧠 **逻辑训练**：强化从“概念 → 数学 → 仿真 → 结论”的完整链路  
- 🧾 **成果留档**：保留读研期间的重要阶段性产出  

---

## ⚠️ 说明

- 本仓库**主要为个人学习与研究使用**  
- 部分内容可能较为草稿化或随理解不断修订  
- 若涉及课程或实验内容，仅作学习记录，不用于任何商业用途
